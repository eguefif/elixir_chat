# Chat

## Description
Learning project. The knowledge goals are:
* Agent
* GenServer
* Tree Supervisor

## Getting Started

### Dependencies

* Elixir

### Executing program

* Server:
```
run mix --no-halt
```

* Client
There is no client. You need to use telenet 127.0.0.1 4040. Here a list of supported commands:
* NAME your_name
* WHISPER interlocutor_name message
